QF-LOG: Quantum Forensic Dataset for QKD Networks

by Madhab Chandra Das · Quantum Forensics Framework

Overview

QF-LOG is a physics-informed forensic dataset for attack detection in Quantum Key Distribution (QKD) networks. It captures realistic behaviour for four scenarios: normal, intercept_resend, detector_blind, and fiber_tap.

50,000 samples (tabular) Time-series LSTM version 4 attack classes QBER · photon_count · latency · abort_flag

Files

  • full_qkd_dataset_realistic_50000.csv – 50k frame-level QKD logs
  • qkd_timeseries_lstm_v2.csv – 1,000 sessions × 50 frames for LSTM / 1D-CNN
  • GitHub repo: QKD-Forensic-Dataset

Feature Schema

qber           float   Quantum Bit Error Rate per frame
photon_count   int     Photon detection count
latency_ms     float   Classical processing latency
abort_flag     int     0 = OK, 1 = Abort
label          string  normal / intercept_resend / detector_blind / fiber_tap

(session-based v2)
session_id     int     QKD session index
frame_index    int     Frame position within the session
